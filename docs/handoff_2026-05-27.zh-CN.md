## ProtoHub / 类似工具判断

- `infranyx/protohub-template` 不是当前需要的抓包工具；它更像 protobuf 仓库/CI 模板。
- 尚未确认视频里真实工具名称，下一步应以“能导出原始 HTTP/WebSocket/gRPC/protobuf/pcap/session”的工具为目标。
- 候选方向：
  - Wireshark：适合 `.pcapng`，支持 Protobuf/gRPC 解析，但通常需要 `.proto` 或 message 类型信息。
  - mitmproxy：适合 HTTP(S)/WebSocket 代理抓包，可保存 flow，后续脚本解析。
  - Fiddler Everywhere：商业 UI 工具，支持 WebSocket/gRPC/Protobuf 调试。
  - BBEye / Fatbeans：中文 Windows 抓包调试工具，可能更接近视频里的用户体验。
  - Proxyman：支持 WebSocket + Protobuf 解码，但主要看具体平台/版本。
  - protobuf-inspector / Protoscope：不是抓包工具，适合对已经拿到的 protobuf bytes 做逆向查看。
