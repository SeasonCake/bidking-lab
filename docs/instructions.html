bidking-lab · 操作说明

玩家向简明手册。推断台暖机或抓屏等待时可在浏览器阅读；关闭本页回到 Streamlit 即可。

这个工具做什么

把游戏信息面板上的读数（紫品总格、金品均价、地图名等）变成约束，再给出出价分布与候选枚举。不会自动出价，也不读取游戏内存。

推荐操作顺序

  1. 侧栏选英雄与地图（先类型后具体地图）
  2. 填写仓库总格数（OCR 通常不会填，请对照游戏手填）
  3. 抓屏 OCR、剪贴板/上传截图，或在「读数输入」手填
  4. 打开顶栏 出价推荐，查看 MC 分布与建议价

四个 Tab

Tab用途
读数输入各品质 cells / 件数 / 总价 / 均格 / 巨物
出价推荐MC 直方图、P25–P90（默认 1500 次）
道具 ROIEthan 默认道具包性价比
联合推断实验性，日常可忽略

抓屏时注意

  • 游戏窗口在前台；信息区在屏幕左侧中部
  • 选对显示器（游戏在哪个屏就选哪个）
  • 首次启动 OCR 需暖机 20–50 秒；同一会话内再次抓屏会更快

常见问题

识别了地图但读数空？
展开侧栏「抓屏 / OCR 诊断」看原文，不清晰就手填。
启动很慢？
首次 OCR 暖机；可在高级里关闭「实屏暖机」后重启 Streamlit。
首次抓屏比第二次慢？
正常现象，同一会话内会更快。
换图后仓库被清空？
请先填仓库格数再抓屏或 OCR 换图。
